## 想说的

> 可能有大佬眼熟，搜索的样式，没错就是仿`Vue`官网的样式，所以后续可能会使用`爬虫 + Flutter3.0/Vue3.0`再开发。

+ 提供思路：
  1. 因为`uTools`一个插件只支持一个页面，要么使用`单页面富应用`，要么使用`<iframe>/<frame>`
  2. 使用`Java/Node`爬虫，爬取相关信息转成`Json`，再读取信息并渲染
